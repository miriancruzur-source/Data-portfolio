# British Airways Reviews Analysis (Tableau)

## Overview
This project analyzes customer reviews of British Airways flights to understand satisfaction levels across different aircraft types, countries, seat classes, and traveler categories.

The analysis was performed using **Tableau** with a focus on rating trends and customer experience metrics.

---

## Tools & Skills Used
- Tableau  
- Data Visualization  
- Exploratory Data Analysis (EDA)  
- Filtering & Interactive Dashboards  

---

## Key Insights
- Identified highest- and lowest-rated **aircraft models**.
- Compared customer satisfaction across **countries** and **traveler types**.
- Analyzed trends for **seat comfort**, **cabin service**, **food quality**, and more.
- Explored year-over-year variation in overall airline ratings.
