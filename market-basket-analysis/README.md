##Project Overview
This project analyzes customer transaction data from the UCI Online Retail Dataset to identify products that are frequently purchased together. Using metrics like **support**, **confidence**, and **lift**, we uncover product associations and propose actionable strategies to increase Average Order Value (AOV) through bundling and cross-selling.

---

##Objectives
- Determine average basket size  
- Identify frequently bought-together product pairs  
- Calculate support, confidence, and lift for item associations  
- Compare purchasing patterns across countries  
- Recommend data-driven product bundles
